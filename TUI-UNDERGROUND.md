## ğŸ› Troubleshooting

### TUI doesn't start

```bash
# Check Node.js version
node --version  # Should be >= 18

# Reinstall dependencies
cd ~/termforum/tui
rm -rf node_modules
npm install
```

### Missing commands

```bash
# Re-run setup
cd ~/termforum
bash setup/termux-ultimate-setup.sh
```

### Theme not applying

```bash
# Set environment variable in .zshrc
echo 'export TERMFORUM_THEME=neon_cyber' >> ~/.zshrc
source ~/.zshrc
```

### Powerlevel10k not working

```bash
# Reconfigure
p10k configure

# Or reinstall
rm -rf ~/.oh-my-zsh/custom/themes/powerlevel10k
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git \
  ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
```
